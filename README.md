# これはkulmsの課題期限取得のchrome拡張機能です

## Chrome拡張機能の設定手順

このプロジェクトをzipファイルとしてダウンロードした後、以下の手順でChrome拡張機能をインストールしてください。

1. **プロジェクトフォルダの解凍**  
   ダウンロードしたzipファイルを解凍し、プロジェクトフォルダ（kulms-helper）を取得します。

2. **Chromeブラウザを開く**  
   Google Chromeブラウザを起動します。

3. **拡張機能ページを開く**  
   アドレスバーに `chrome://extensions/` と入力して拡張機能管理ページを開きます。

4. **デベロッパーモードを有効にする**  
   ページ右上の「デベロッパーモード」のスイッチをオンにします。

5. **拡張機能を読み込む**  
   「パッケージ化されていない拡張機能を読み込む」ボタンをクリックします。

6. **フォルダを選択**  
   解凍したプロジェクトフォルダ（kulms-helper）を選択して開きます。

7. **インストール完了**  
   拡張機能が読み込まれ、拡張機能リストに表示されます。必要に応じて有効化してください。

これで、kulmsの課題期限取得機能が利用可能になります。
